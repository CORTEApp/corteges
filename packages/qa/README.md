# QA

Objetivo de la V1:
- revisar bloques en aislamiento
- revisar composición de páginas
- detectar regresiones visuales
- validar reglas básicas de accesibilidad

## Estrategia

1. Storybook para bloques
2. Pruebas de interacción en stories críticas
3. Playwright para snapshots y rutas clave
4. Informe legible para el agente `DesignQA`
